# 🛡️ 文字修復大作戰 (Word Repair Challenge)

這是一款專為 **國小四年級學生** 設計的互動語文學習遊戲。透過扮演「文字修復工程師」，學生將挑戰三道關卡，修復被「語文病毒」感染的系統，最後由 AI 導師生成專屬的修復報告。

## 🌟 遊戲特色

- **趣味情境**：將語文練習轉化為「修復任務」，提升學習動機。
- **AI 互動**：最後關卡與 Gemini AI 連動，提供個人化、鼓勵性的結案報告。
- **隨機題庫**：三大關卡皆設有隨機抽題機制，每次挑戰都有新鮮感。
- **無圖設計**：專注於純文字的辨析與邏輯，訓練細微的觀察力。

---

## 🧩 關卡介紹

### 第一關：字魔捉迷藏 (字形辨析)
系統會出現一段含有 2 個錯別字的短文，學生需找出受感染的文字（如：留與流、惢與蕊）。
* **偵探提示**：卡關時可點擊提示，錯字會自動加粗變紅。

### 第二關：成語天秤儀 (語意邏輯)
提供特定的生活情境，學生必須從三個選項中，選出「成語 + 解釋」完全正確的組合。
* **陷阱設計**：部分選項可能成語正確但解釋錯誤，考驗細心程度。

### 第三關：標點符號定身術 (斷句練習)
面對一段完全沒有標點的文字（如：下雨天留客天留我不留），學生需提交兩種截然不同的斷句方式，體驗標點符號改變語意的魔力。

## 🛠️ 技術架構

- **開發語言**：HTML5, CSS3, JavaScript (Frontend) / Google Apps Script (Backend)
- **AI 模型**：Google Gemini 1.5 Flash
- **安全機制**：使用 `PropertiesService` 隱藏 API Key，保護開發者資訊。

---

## 📝 安裝與部署 (給開發者)

1. 建立一個 Google Apps Script 專案。
2. 建立 `Code.gs` 並貼入後端代碼。
3. 建立 `index.html` 並貼入前端代碼。
4. 在 GAS 「專案設定」中新增指令碼屬性 `GEMINI_API_KEY`。
5. 點擊「部署」>「網頁應用程式」，並設定為「任何人」皆可存取。
